#Angular V5 & Angular CLI - Eine Übersicht

## Voraussetzungen und Vorbereitungen

Node (NodeJS + NPM) muss an Ihrem Betriebssystem installiert sein. Das können Sie am einfachsten überprüfen, wenn Sie im Terminal folgendes ausführen:

```
node -v 
```
