#Array Functions

| Befehl | Beschreibung |
|:--- |:---|
| `concat` | führt mehrere Arrays zusammen |
| `join('TRENNZEICHEN')` | fügt Array-Elemente zusammen zu einem Array |
| `slice (von, bis?)` | schneidet Elemente, Original-Array bleibt unberührt |
| `splice (von, bis?)` | schneidet Original-Array |
| `filter` | `var filtered = array.filter((item) => item.x === 'BEDINGUNG'` Sofern die Bedingung `true` ist, wird das jeweilige Element retourniert) |
